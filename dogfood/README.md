# dogfood

real programs written in pyr. rebuilt versions of unix tools and utilities, used to stress-test the language and find rough edges.

## projects

- [grep](grep/) - search files for patterns, print matching lines
- [wc](wc/) - count lines, words, and bytes
- [cat](cat/) - concatenate and print files
- [head](head/) - print the first N lines of a file
- [jq](jq/) - JSON query and transform (subset)
- [httpd](httpd/) - static file HTTP server
